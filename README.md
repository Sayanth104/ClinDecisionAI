# 🏥 ClinDecisionAI

ClinDecisionAI is an Explainable AI-based clinical decision support system designed to predict patient risk levels using machine learning models and provide interpretable insights through SHAP/LIME along with a web-based dashboard.

---

## 🚀 Features

* 🔍 Predicts patient risk levels (Low, Medium, High)
* 🧠 Uses machine learning models (Random Forest / SVM)
* 📊 Provides explainability using SHAP / LIME
* 📈 Interactive dashboard for visualization
* 📄 Generates structured clinical reports

---

## 🧰 Tech Stack

Frontend: React / Next.js, Tailwind CSS
Backend: Python
Machine Learning: Scikit-learn, SHAP / LIME

---

## 📊 Input Parameters

* Age
* Blood Pressure
* Cholesterol
* Glucose
* BMI

---

## ▶️ How to Run

### 1. Clone the repository

git clone https://github.com/Sayanth104/ClinDecisionAI.git
cd ClinDecisionAI

---

### 2. Run Frontend (Next.js)

npm install
npm run dev

Frontend runs at: http://localhost:3000

---

### 3. Run Backend (Python)

cd backend
pip install -r requirements.txt
python main.py

---

## ⚠️ Note

Make sure both frontend and backend are running at the same time for full functionality.
