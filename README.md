# 🏥 ClinDecisionAI

ClinDecisionAI is an Explainable AI-based clinical decision support system designed to predict patient risk levels using machine learning models and provide interpretable insights through SHAP/LIME along with a web-based dashboard.

---

## 🚀 Features

* 🔍 Predicts patient risk levels (Low, Medium, High)
* 🧠 Uses machine learning models (Random Forest / SVM)
* 📊 Provides explainability using SHAP / LIME
* 📈 Interactive dashboard for visualization
* 📄 Generates structured clinical reports

---

## 🧰 Tech Stack

Frontend: React / Next.js, Tailwind CSS
Backend: Node.js
Machine Learning: Python, Scikit-learn, SHAP / LIME

---

## 📊 Input Parameters

* Age
* Blood Pressure
* Cholesterol
* Glucose
* BMI
