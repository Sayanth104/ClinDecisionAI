## Technical Architecture

### Frontend Stack
- **Framework**: Next.js 16 with App Router
- **UI Components**: shadcn/ui with Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **State Management**: React Hooks with useState/useEffect
- **Data Storage**: localStorage for authentication
- **Styling**: Tailwind CSS v4 with custom design tokens (medical blue theme)

### Mock AI Logic
- Risk Score Calculation: Weighted formula combining patient metrics
- SHAP Feature Importance: Real-time calculation of feature contributions
- Similar Patient Cases: KNN-style mock matching with similarity scores
- Risk Classification: Automatic categorization based on score thresholds

### Data Flow
1. User inputs patient data on Prediction page
2. Risk score calculated in real-time using mock ML formula
3. SHAP values computed based on feature weights
4. Similar patient cases generated
5. Reports created with full patient context
6. All data persisted in component state
